About the GitHub foundational certification

This exam recognizes your ability to demonstrate the following foundational competencies on GitHub.

  • Foundational Git and GitHub concepts
  • Components of modern software development practices using GitHub
  • GitHub distribution and consumption models
  • GitHub security, compliance, privacy and trust options
  • The benefits of the GitHub community
